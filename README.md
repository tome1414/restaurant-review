# Restaurant Review Flow

飲食店向けのレビュー促進ツールのプロトタイプ・モックアップです。  
QRコードをテーブルに設置するだけで、お客様のレビュー投稿を自然な形で促進できます。

## 概要

来店客がQRをスキャンすると、以下のフローでレビュー投稿まで案内されます。

```
QRスキャン
  ↓
① 言語選択（自動検出）
  ↓
② 満足度評価（内部フィルタリング）
  ├─ 4★以上 → ③ プラットフォーム選択 → ④ AI文章生成 → ⑤ 投稿手順ガイド
  └─ 3★以下 → 内部フィードバックフォーム（外部レビューには流さない）
  ↓
⑥ ルーレット抽選（特典プレゼント）
  ↓
⑦ LINE公式アカウント 友達追加
```

## 主な機能

### 🔍 内部フィルタリング
低評価（1〜3★）はGoogleなどの外部レビューに流さず、店舗へのフィードバックとして吸収。高評価のみ外部レビューへ誘導することで、評価スコアを健全に保ちます。

### 🌐 多言語・プラットフォーム最適化
お客様のスマートフォンの言語設定を自動検出し、最も効果的なレビューサイトへ誘導を分岐。

| 言語 | 誘導先（優先順） |
|------|----------------|
| 日本語 | Google マップ / Instagram |
| タイ語 | Wongnai / Google マップ |
| 英語・その他 | TripAdvisor / Google マップ |

オーナーの希望に合わせてカスタマイズ可能です。

### ✨ AI口コミ文章生成
アンケート回答をもとにAIが口コミ文章案を自動生成。お客様はコピー＆ペーストするだけで投稿でき、レビューへの心理的ハードルを大幅に下げます。

### 🎰 ガチャ／ルーレット型インセンティブ
回答後にデジタルルーレットを回して特典が当たる仕組み。固定の特典より参加率・エンゲージメントが向上します。

例：1等「お会計20%オフ」／2等「デザート無料」／3等「次回ドリンク券」

### 💚 LINE公式アカウント統合
当選クーポンをLINEで保存することで友達リストを獲得。その後のリマーケティング（クーポン配信・新メニュー告知・誕生日クーポン）につなげます。

## 想定ユースケース

- 飲食店（レストラン・カフェ・居酒屋など）
- テーブルへのQRコード設置
- レビュー代行業者・MEO対策事業者のホワイトラベル利用

## デモ

`index.html` をブラウザで開くか、ローカルサーバーで確認できます。

```bash
python3 -m http.server 3000
# → http://localhost:3000
```

ナビゲーションバーで各画面を切り替えながら確認できます。
